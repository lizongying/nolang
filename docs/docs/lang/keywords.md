### 運算符

#### 特殊符號

- ... // 返回語句，终止函数 替代return
- #   // 引入模塊
- ->  // 類型轉換
- .   // 代表自身，類或參數
- ..  // 父類
- <-  // for i <- [1,2, 3]

未來會以下替代：

- !{} // for
- *   // continue 
- **  // break

#### 算術運算符

```shell
+   // 加法
-   // 減法
*   // 乘法
/   // 除法
```

#### 比較運算符

```shell
==  // 等於
!=  // 不等於
<   // 小於
>   // 大於
<=  // 小於等於
>=  // 大於等於
```

#### 邏輯運算符

```shell
&&  // 邏輯與
||  // 邏輯或
!   // 邏輯非
```
