### 運算符

#### 特殊符號

- `...` // 返回語句，终止函数 替代return
- `#` // 引入模塊
- `->` // 類型轉換
- `<-` // for i <- [1,2, 3] 替代in
- `?` // 空 替代nil
- `..` // 父類
- `.` // 真。代表自身，類或參數 替代true
- `!` // 假，錯誤 替代false
- `!{}` // 循環 替代for while loop
- `*` // 繼續 替代continue
- `**` // 中斷 替代break

#### 算術運算符

- `+` // 加法
- `-` // 減法
- `*` // 乘法
- `/` // 除法

#### 比較運算符

- `==` // 等於
- `!=` // 不等於
- `<` // 小於
- `>` // 大於
- `<=` // 小於等於
- `>=` // 大於等於

#### 邏輯運算符

- `&&` // 邏輯與
- `||` // 邏輯或
- `!` // 邏輯非
